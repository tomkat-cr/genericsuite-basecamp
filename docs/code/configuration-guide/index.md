# GenericSuite App Creation and Configuration guide

### Validation

The validation configuration defines the validation rules for the CRUD editor.

* The TypeScript interfaces to validate the frontend JSON configurations are defined in the [CrudEditorConfigInterface.ts](./CrudEditorConfigInterface.ts) file.

* The Python classes to validate the frontend JSON configurations are defined in the [crud_editor_config_classes.py](./crud_editor_config_classes.py) file.
