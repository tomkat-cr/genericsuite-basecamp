# La IA de GenericSuite

![gs_ai_logo_circle.png](../assets/images/gs_ai_logo_circle.png)

La **GenericSuite AI** es una extensión de [GenericSuite Core](./genericsuite-core.md) enfocada en el desarrollo de aplicaciones de IA.

Tiene los siguientes elementos:

* :fontawesome-brands-react:{ .react } [GenericSuite AI (versión frontend) para React.js](./Frontend-Development/GenericSuite-AI/index.md)
* :fontawesome-brands-python:{ .python } [GenericSuite AI (versión de backend) para Python](./Backend-Development/GenericSuite-AI/index.md)
* :fontawesome-brands-linux:{ .linux } [GenericSuite Scripts (versión de backend)](./Backend-Development/GenericSuite-Scripts/index.md)

Consulta la [Guía de Configuración](./Configuration-Guide/index.md) para aprender cómo desarrollar una aplicación, configurando los editores CRUD, puntos finales, menús, asistentes de IA y más.
