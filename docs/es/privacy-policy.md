# Política de Privacidad

**Fecha de entrada en vigor:** 1 de enero de 2024

## Introducción

GenericSuite ("nosotros", "nuestro" o "nos") es una biblioteca de software de código abierto diseñada para agilizar los flujos de trabajo de frontend y backend. Esta Política de Privacidad explica cómo manejamos la información en relación con la biblioteca de software GenericSuite y nuestro sitio web de documentación.

## 1. Información que Recogemos

### A. La Biblioteca GenericSuite
La biblioteca de software GenericSuite en sí no recopila, almacena ni transmite datos personales de sus usuarios o de los usuarios finales de las aplicaciones construidas con ella. Es un conjunto de utilidades y patrones de codificación que "viven" dentro de su propia infraestructura.

### B. Sitio web de Documentación
Cuando visites nuestro sitio web de documentación (p. ej., genericsuite.carlosjramirez.com o genericsuite.readthedocs.io), es posible que recopilemos datos de uso anónimos a través de registros estándar del servidor web o análisis de terceros (como Google Analytics o ReadTheDocs analytics) para ayudarnos a mejorar la documentación. Estos datos pueden incluir:
- Tipo y versión del navegador
- Sistema operativo
- Sitio web de referencia
- Páginas visitadas y tiempo dedicado

## 2. Cómo Usamos la Información
Toda la información recopilada a través del sitio web de documentación se utiliza exclusivamente para:
- Analizar tendencias y participación de los usuarios.
- Mejorar el contenido y la estructura de nuestra documentación.
- Mantener la seguridad y el rendimiento del sitio.

## 3. Servicios de Terceros
GenericSuite puede integrarse con servicios de terceros (p. ej., OpenAI, Google Cloud, AWS) si elige configurarlos. No tenemos acceso a los datos que envía a estos servicios. Usted es responsable de revisar las políticas de privacidad de cualquier servicio de terceros que use junto con GenericSuite.

## 4. Actualizaciones de Esta Política de Privacidad
Podemos actualizar esta Política de Privacidad de vez en cuando. Le notificaremos cualquier cambio publicando la nueva Política de Privacidad en esta página.
