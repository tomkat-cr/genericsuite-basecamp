# Código de ejemplo

En esta sección puedes encontrar código de muestra para ayudarte a empezar con GenericSuite.

## Aplicación de ejemplo

[![Banner de ExampleApp](../../code/exampleapp/assets/exampleapp_banner_01.png)](../../code/exampleapp/README.md)    

[EjemploApp](../../code/exampleapp/README.md) es una aplicación de ejemplo completa, construida como un monorepo usando Turborepo y pnpm. Esto proporciona un plano práctico, del mundo real, para que los desarrolladores aprendan de él y aceleren sus propios proyectos. Hay un frontend en React y backends en Python, usando los tres marcos principales: FastAPI, Flask y Chalice.

## Plantilla de FastAPI

[![Banner de FastAPI Template](../../code/fastapitemplate/assets/fastapitemplate_banner_01.png)](../../code/fastapitemplate/README.md)    

[Plantilla de FastAPI](../../code/fastapitemplate/README.md) es una plantilla de aplicación full-stack que te permite tener una base para comenzar a desarrollar tu aplicación basada en GenericSuite. Se ha construido como un monorepo usando workspaces de npm, con un frontend en React y un backend en Python, el [framework FastAPI](https://fastapi.tiangolo.com/).
