# Privacy Policy

**Effective Date:** January 1, 2024

## Introduction

GenericSuite ("we," "our," or "us") is an open-source software library designed to streamline frontend and backend workflows. This Privacy Policy explains how we handle information in relation to the GenericSuite software library and our documentation website.

## 1. Information We Collect

### A. The GenericSuite Library
The GenericSuite software library itself does not collect, store, or transmit any personal data from its users or the end-users of applications built with it. It is a set of utilities and coding patterns that "live" within your own infrastructure.

### B. Documentation Website
When you visit our documentation website (e.g., genericsuite.carlosjramirez.com or genericsuite.readthedocs.io), we may collect anonymous usage data through standard web server logs or third-party analytics (like Google Analytics or ReadTheDocs analytics) to help us improve the documentation. This data may include:
- Browser type and version
- Operating system
- Referring website
- Pages visited and time spent

## 2. How We Use Information
Any information collected via the documentation website is used solely for:
- Analyzing trends and user engagement.
- Improving the content and structure of our documentation.
- Maintaining the security and performance of the site.

## 3. Third-Party Services
GenericSuite may integrate with third-party services (e.g., OpenAI, Google Cloud, AWS) if you choose to configure them. We do not have access to the data you send to these services. You are responsible for reviewing the privacy policies of any third-party services you use in conjunction with GenericSuite.

## 4. Updates to This Policy
We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.

## 5. Contact Us
If you have any questions about this Privacy Policy, please contact us via our GitHub repository: [https://github.com/tomkat-cr/genericsuite-basecamp](https://github.com/tomkat-cr/genericsuite-basecamp).
